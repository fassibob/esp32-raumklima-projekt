# esp32-raumklima-projekt
Messung von Temperatur, Luftfeuchtigkeit und CO2 mit ESP32
Wir wollen in den nächsten Tagen gemeinsam ein Cyber-Physisches System einrichten, das 
Luftfeuchtigkeit
Temperatur und
CO2-Gehalt der Luft misst. Die Ergebnisse könenn über Webbrowser und ein kleines Display ausgegeben werden.
